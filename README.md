## Job Portal

### Getting Start Guide

```bash
git clone project_url

cd project_folder

composer install

cp .env.example .env

php artisan key:generate

# Set Up APP_URL the Database

php artisan migrate

php artisan db:seed

npm install

npm run dev
```
